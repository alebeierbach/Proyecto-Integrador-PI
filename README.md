El trabajo consiste en el diseño e implementación de un prototipo de granja urbana vertical
inteligente a escala reducida, con tres niveles de cultivo en sustrato. El sistema contará con sensado
ambiental (temperatura, humedad, luz), riego por goteo con recirculación, iluminación LED full
espectro, ventilación activa y sistemas para el control de los factores ambientales (sol, lluvia, frío,
calor). El control se realizará mediante microcontrolador, con adquisición y análisis de datos,
orientado a mejorar la eficiencia hídrica y energética.
De forma complementaria, se encuentra en curso otro Proyecto Integrador enfocado en el diseño de
una red de transmisión de datos para la comunicación de estos sistemas. Si bien ambos trabajos
comparten el contexto temático, el presente PI se centra exclusivamente en el d iseño, armado y
control automatizado del prototipo de cultivo, manteniendo autonomía funcional respecto del
desarrollo de redes.
